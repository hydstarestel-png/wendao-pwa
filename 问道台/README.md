# 问道台 · 考研修炼系统

本地使用时双击 `启动问道台.cmd`。发布到 HTTPS 后可在手机浏览器中安装为 PWA；连接云端账号后，电脑和手机共用同一份修行档案。

## 已实现

- 《仙逆》凝气十五层至问鼎的修为体系；凝气每层 320 修为，总计 25,000 修为问鼎。
- 今日修炼令支持逐单元即时增加/撤回修为；深度、标准、轻度修炼分别计 12、10、6 修为，每日封顶 18 单元与 180 修为。
- 自定义修炼任务可设置类型、目标单元和开始时间，并参与同一套即时计分。
- 支持拍摄或上传课表、计划表和清单图片，在设备本地完成中英文 OCR，预览修正后批量生成同格式任务。
- 守时、清心、强体、定神四枚道基印，以及“修为 + 试炼”的双钥匙晋级。
- 四周渐进学习量、六日修炼一日安息、三阶段手机与游戏额度。
- 最低保命日、每日任务到期提示、站内提醒与页面开启时的浏览器通知。
- 英语入宗四试、英语道境天梯、两段式英语日课。
- 已同步英语入宗试炼一：剑桥雅思20 Test 1 为 33/40；正式道境为元婴圆满，英语修为 45/1800 E-XP。
- 听脉精炼、神识续航、言灵输出、原著炼心与雅思试炼可直接计入 E-XP。
- 浏览器本地时间每 15 秒自动校准；日期、倒计时、逾期任务与跨日任务会自动刷新，无需手动重载。
- 今日复盘与反思、七日趋势、历史修行簿、二度复盘，以及 JSON 数据导入导出。
- PWA 安装、离线应用外壳、本地优先存储，以及基于账号与行级权限的多端云同步。

## 隐私与限制

图表图片不上传，仅在设备内识别；首次识别需要联网下载 OCR 模型。浏览器通知仍要求页面或已安装的 PWA 处于可运行状态。云同步需要先按 `DEPLOYMENT.md` 配置 Supabase。
